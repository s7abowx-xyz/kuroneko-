الشروط والأحكام

1. استخدام الخدمة

استخدامك لواجهة Alyacore يعني موافقتك على هذي الشروط. الخدمة مقدّمة "كما هي" بدون أي ضمانات.

2. مفتاح API

مفتاح API الخاص بك مسؤوليتك الشخصية. لا تشاركه مع أحد، ولك حرية تجديده بأي وقت من صفحة حسابك.

3. الاستخدام المقبول

ممنوع استخدام الخدمة لأي غرض غير قانوني، أو لانتهاك حقوق ملكية فكرية، أو لإرسال محتوى ضار.

4. الاشتراكات

خطة Pro اشتراك شهري متجدد عبر Stripe. تقدر تلغيه بأي وقت من صفحة إدارة الاشتراك.

5. التعديلات

هذي الشروط قابلة للتعديل بأي وقت. استمرارك باستخدام الخدمة بعد أي تعديل يعني موافقتك عليه.
